Detailed Description

template<typename WindowingType>
struct Carna::qt::WindowingControl::GenericWindowingAdapter< WindowingType >

Implements the Windowing interface as an adapter to WindowingType in terms of compile-time polymorphism.

The unspecialized version expects WindowingType to support the following methods in UML notation:

  • setWindowingLevel(level : base::HUV)
  • setWindowingWidth(width : unsigned int)
  • windowingLevel() : base::HUV
  • windowingWidth() : unsigned int

Specialize the template to overcome this expectations.
